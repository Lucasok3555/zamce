# Zamce - Mini Navegador em Python

Aplicação de navegador com interface gráfica criada em **Python + Qt WebEngine**.

## Recursos
- Renderização real de páginas com **HTML, CSS e JavaScript**.
- Barra de navegação com voltar, avançar, recarregar e home.
- Campo inteligente: aceita URL direta ou texto para busca.

## Requisitos
- Python 3.10+
- Pacotes:
  - `PyQt6`
  - `PyQt6-WebEngine`

## Instalação
```bash
pip install PyQt6 PyQt6-WebEngine
```

## Executar
```bash
python mini_navegador.py
```

## Observação
O arquivo `server.js` permanece no repositório como utilitário separado e não é usado pelo navegador Python.
